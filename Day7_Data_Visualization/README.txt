Titanic Dataset Visualization Project

This project focuses on data visualization and exploratory data analysis (EDA) using the Titanic Dataset dataset.

The main objective of this project is to understand how different visualization techniques help analyze and interpret data patterns, relationships, and distributions.

The project was completed using a Jupyter Notebook (.ipynb) with Python visualization libraries.

Dataset used:
Titanic Dataset on Kaggle

The dataset contains information about Titanic passengers, including:

Passenger class
Gender
Age
Fare
Survival status
Family information
Embarked port


Technologies Used
Python
Jupyter Notebook
Pandas
NumPy
Matplotlib
Seaborn



Visualizations Used
1. Bar Charts

Bar charts were used to compare categorical data such as:

Survival count
Survival by gender
Passenger class distribution
Survival by passenger class
Purpose: 
To compare categories and identify trends in survival rates and passenger information.

2. Histograms

Histograms were used to analyze numerical distributions such as:

Age distribution
Fare distribution
Purpose

To understand the spread, frequency, and distribution of continuous variables.

3. Scatter Plots

Scatter plots were used to visualize relationships between variables such as:

Age vs Fare
Age vs Fare by Survival
Purpose

To observe relationships, correlations, clusters, and outliers.

4. Heatmap

A correlation heatmap was created using numerical features in the dataset.

Purpose

To identify relationships and correlations between variables.

5. Pair Plot

Pair plots were used to visualize relationships between multiple numerical features simultaneously.

Purpose

To explore patterns, distributions, and feature interactions efficiently.

Key Observations
Female passengers had a higher survival rate than male passengers.
First-class passengers survived more frequently compared to lower classes.
Most passengers were between 20–40 years old.
Fare distribution was highly skewed with some extreme values.
Passenger class showed a noticeable relationship with fare and survival.
