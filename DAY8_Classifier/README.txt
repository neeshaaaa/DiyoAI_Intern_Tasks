Iris Classification using Scikit-Learn

Project Overview:
This project demonstrates a beginner-friendly machine learning classification workflow using the Iris dataset and the Scikit-Learn library in Python.
The goal of this project is to understand the basic concepts of machine learning by building and evaluating classification models on a real dataset.

The project includes:

Data loading and exploration
Data visualization
Train-test splitting
Model training
Prediction and evaluation
Accuracy comparison of multiple classifiers
Dataset

This project uses the built-in Iris dataset available in Scikit-Learn.

The dataset contains measurements of iris flowers from three different species:

Setosa
Versicolor
Virginica
Features

The dataset includes the following features:

Sepal Length
Sepal Width
Petal Length
Petal Width
Target

The target variable is the flower species.
