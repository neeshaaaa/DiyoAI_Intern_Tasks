# Text Statistics Analyzer (Python)

# Project Description
A simple Python project that analyzes a text file and generates useful statistics like word count, sentence count, unique words, and most frequent words.



# Features
- Count total words
- Count total sentences
- Count unique words
- Find top 10 most frequent words
- Clean and preprocess text



# Technologies Used
- Python
- Jupyter Notebook
- Collections (Counter)
- String handling
