# Student Exam Data Analysis Project

This project is a beginner-friendly data analysis project built using Python.  
It demonstrates how to work with:

- NumPy arrays
- Pandas DataFrames
- CSV files

The project analyzes student exam marks, calculates totals and percentages, filters high scorers, and generates a final report.

---------------------------------------------------

# What is NumPy?
NumPy (Numerical Python) is a Python library used for numerical and mathematical operations.

It provides support for:
- Arrays
- Matrices
- Mathematical functions
- Statistical operations

NumPy is faster and more efficient than normal Python lists for numerical computations.

--------------------------------------------------

## Why We Use NumPy

We use NumPy because it:
- Handles large numerical data efficiently
- Performs fast mathematical calculations
- Supports array operations
- Provides statistical functions like:
  - mean()
  - max()
  - min()
  - sum()

---------------------------------------------------
What is Pandas?
Pandas is a Python library used for data analysis and data manipulation.

It provides two important data structures:

1. Series
2. DataFrame

A DataFrame is similar to a table with rows and columns.

Why We Use Pandas?

We use Pandas because it:

Makes data handling easy
Reads CSV files quickly
Organizes data in tabular format
Supports filtering and sorting
Performs data analysis efficiently

--------------------------------------------------

What is a CSV File?
CSV stands for Comma-Separated Values.

It is a simple file format used to store tabular data.

Why We Use CSV Files?

CSV files are widely used because they:

Are simple and lightweight
Can be opened in Excel
Store structured data easily
Are supported by many programming languages
Help transfer data between applications

In this project, CSV files are used to:

Save student records
Read stored data
Generate reports
